multi-php-git-deploy
===

Lightweight and highly configurable script to deploy multiple PHP applications using Git. 
Small/Fast script that helps deploying multiple PHP applications with Git.

This project webstyle and code is based upon this: https://github.com/markomarkovic/simple-php-git-deploy

MD, short for MultiDeploy, consist of a main file and a unlimited number of config file. One for each site you want to deploy.
